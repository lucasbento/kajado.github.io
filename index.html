Kajado
Cada projeto é único, todos com suas diferenças e similaridades, a ideia do Kajado é dar a possibilidade de total personalização dos componentes, muito além das bibliotecas e frameworks atuais, com o kajado você é quem cria os componentes que necessita, totalmente em português.

Em portugues

Todas as classes são escritas em PT-BR, o que torna o Kajado uma biblioteca de fácil aprendizagem.

Open Source

Use como quiser, a biblioteca kajado é 100% open source, sob a licença MIT.

Responsivo

Faça aplicações para os mais diversos dispositivos usando as classes responsivas.

Estilo Fluente

Crie seus elementos usando classes semânticas que definem as caracteristicas desejadas.
